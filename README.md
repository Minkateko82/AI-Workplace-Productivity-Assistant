 
# AI Workplace Productivity Assistant

## Project Overview
This project is an AI-powered workplace productivity assistant designed to help professionals automate everyday tasks and improve productivity using AI.

## Features
- Smart Email Generator with tone selection (formal, friendly, persuasive)
- Meeting Notes Summarizer that extracts key points and action items
- AI Task Planner for daily and weekly task prioritization
